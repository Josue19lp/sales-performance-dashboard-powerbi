# 📊 Dashboard de Análisis de Datos en Power BI

## 📌 Descripción del proyecto

Este proyecto consiste en un dashboard interactivo desarrollado en Power BI, conectado a una base de datos local, con el objetivo de analizar información relevante.

El sistema transforma datos en visualizaciones dinámicas que permiten identificar tendencias, patrones y métricas clave.

---

## 🎯 Objetivo

- Analizar datos almacenados en una base de datos local
- Diseñar un dashboard interactivo en Power BI
- Aplicar modelado de datos y relaciones
- Generar visualizaciones para toma de decisiones

---

## 🧠 Fuente de datos

- Base de datos local (SQL Server)
- Datos estructurados para análisis de métricas

---

## 📊 Visualizaciones incluidas

- KPIs principales
- Tendencias temporales
- Comparativas de categorías
- Gráficos dinámicos
- Filtros interactivos

---

## 🛠️ Herramientas utilizadas

- Power BI
- SQL Server

---

## 📁 Archivos del proyecto

- `.pbix` → archivo del dashboard
- `database.sql` → estructura de la base de datos (La base de datos original no se incluye debido a su tamaño)
- screenshots del proyecto
